Project Overview

Project Description

This project investigates the use of surface electromyography (sEMG) signals captured from facial and neck muscles to recognise silent or mouthed speech — enabling human communication without producing audible sound. Raw bio-signals are processed, features extracted, and speech commands classified using machine learning.

About Me

I am Arambegedara A.D.D.B, a third year Engineering undergraduate at the University of Peradeniya (Student ID: E/21/306). This portfolio documents all activities and materials I personally contributed to, supervised by Dr. Ruwan Ranaweera.

Research Objectives

  • Acquire clean sEMG signals from speech-related muscles
  • Build a robust preprocessing and noise filtering pipeline
  • Extract discriminative features for ML classification
  • Achieve high-accuracy silent speech word recognition
  • Evaluate the real-time feasibility of the system

Technologies & Tools
